# FLAPI
(Entwurf) API zur Filmliste
## Setup
```
git clone git@github.com:mediathekview/FLAPI.git FLAPI
cd FLAPI
composer install
composer start
```

## Documentation
### REST
```
npm install apidoc -g
composer docs
```
Docs werden dann in `docs/` gespeichert.

###PHP
```
composer api-docs
```
Docs werden dann in `docs/api/` gespeichert.
